# Xylophone App

The Xylophone app is a simple and fun music app developed using Flutter. It allows users to play various musical notes by tapping on colorful bars displayed on the screen, mimicking the functionality of a xylophone instrument.

![Bitcoin Ticker App Demo](xylophone.mp4)

## Features

- Easy-to-use interface: Users can simply tap on the colored bars to produce musical notes.
- Multiple notes: Each colored bar corresponds to a different musical note, providing a wide range of sounds for users to experiment with.
- Realistic sounds: The app generates realistic xylophone sounds, creating an immersive musical experience.
- Colorful design: The vibrant colors and sleek design make the app visually appealing and engaging for users of all ages.

## Installation

1. Clone the repository to your local machine:

   ```
   git clone https://github.com/zahidprvz/xylophone-app.git
   ```

2. Navigate to the project directory:

   ```
   cd xylophone-app
   ```

3. Install dependencies using Flutter:

   ```
   flutter pub get
   ```

4. Run the app on your preferred device or simulator:

   ```
   flutter run
   ```

## Usage

- Tap on the colored bars to play different musical notes.
- Experiment with tapping different combinations of bars to create your own melodies and tunes.
- Enjoy the soothing sounds of the xylophone and let your creativity flow!
